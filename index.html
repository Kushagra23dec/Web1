Can there be a paht longer than Hamiltonian path?Why?

No, because Hamiltonian path is the only path in a graph that covers all the vertices without repeating any edge and vertex, since a path is a walk without repeating any of the vertices and edges, therefore we can say that there is no path longer than hamiltonian path.
